Class Inputfield

An Inputfield an actual form input field widget, and this is provided as the base class for different kinds of form input widgets provided as modules.

The class supports a parent/child hierarchy so that a given Inputfield can contain Inputfields below it. An example would be the relationship between fieldsets and fields in a form.

An Inputfield is typically associated with a Fieldtype module.

All Inputfields have the following properties at minimum:

Inputfield::id A unique string identifier for this Inputfield

Inputfield::class Class name(s) to be rendered with the Inputfield XHTML output

Inputfield::name Corresponds to XHTML "name" attribute

Inputfield::value The current value of the field. May correspond go the XHTML "value" attribute on some inputs.

Wire implements WireTranslatable, WireHookable, WireFuelable, WireTrackable
WireData implements IteratorAggregate
Inputfield implements Module

Direct known subclasses

InputfieldCheckbox, InputfieldCommentsAdmin, InputfieldPageListSelect, InputfieldPageListSelectMultiple, InputfieldPageTable, InputfieldRepeater, InputfieldSelect, InputfieldSelector, InputfieldSubmit, InputfieldText, InputfieldWrapper, InputfieldDatetime, InputfieldFieldsetClose, InputfieldFile, InputfieldHidden, InputfieldIcon, InputfieldInteger, InputfieldPage, InputfieldPageAutocomplete

Indirect known subclasses

InputfieldAsmSelect, InputfieldButton, InputfieldImage, InputfieldMarkup, InputfieldName, InputfieldPageName, InputfieldPageTitle, InputfieldPassword, InputfieldRadios, InputfieldSelectMultiple, InputfieldTextarea, InputfieldURL, InputfieldCheckboxes, InputfieldCKEditor, InputfieldEmail, InputfieldFieldset, InputfieldFieldsetOpen, InputfieldFieldsetTabOpen, InputfieldFloat, InputfieldForm
