ProcessWire Repeater Fieldtype

Maintains a collection of fields that are repeated for any number of times.

For documentation about how Fieldtypes work, see: /wire/core/Fieldtype.php /wire/core/FieldtypeMulti.php

ProcessWire 2.x Copyright (C) 2015 by Ryan Cramer This file licensed under Mozilla Public License v2.0 http://mozilla.org/MPL/2.0/

https://processwire.com

Wire implements WireTranslatable, WireHookable, WireFuelable, WireTrackable
WireData implements IteratorAggregate
Fieldtype implements Module
FieldtypeRepeater implements ConfigurableModule
