Class ProcessPageLister

Process Lister __ _ __ / / (_)____/ /____ _____ / / / / ___/ __/ _ \/ ___/ / /___/ (__ ) /_/ __/ / /_____/_/____/\__/\___/_/

Provides an alternative listing view for pages using specific templates.

ProcessWire 2.8.x, Copyright 2016 by Ryan Cramer https://processwire.com

For support of actions, new edit modules, and custom configurable Listers, check out ListerPro at http://processwire.com/ListerPro/

GET vars recognized by Lister:

session_bookmark: name of session bookmark (session_bookmark=...) pageNum: page number, if specified as GET var rather than URL segment (pageNum=1) open: CSV string of page IDs that should be open/selected (open=1,2,3) reset: Initiates reset of filters (reset=1) minimal: Minimal mode shows only results and nothing else (minimal=1, usually combined with modal=1)

PROPERTIES

Wire implements WireTranslatable, WireFuelable, WireTrackable
WireData implements IteratorAggregate, ArrayAccess
Process implements Module
ProcessPageLister implements ConfigurableModule
