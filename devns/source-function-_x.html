138: /**
139:  * Send an email or retrieve the mailer object
140:  *
141:  * Note 1: The order of arguments is different from PHP's mail() function.
142:  * Note 2: If no arguments are specified it simply returns a WireMail object (see #4 below).
143:  *
144:  * This function will attempt to use an installed module that extends WireMail.
145:  * If no module is installed, WireMail (which uses PHP mail) will be used instead.
146:  *
147:  * This function can be called in these ways:
148:  *
149:  * 1. Default usage:
150:  *
151:  *    wireMail($to, $from, $subject, $body, $options);
152:  *
153:  *
154:  * 2. Specify body and/or bodyHTML in $options array (perhaps with other options):
155:  *
156:  *    wireMail($to, $from, $subject, $options);
157:  *
158:  *
159:  * 3. Specify both $body and $bodyHTML as arguments, but no $options:
160:  *
161:  *    wireMail($to, $from, $subject, $body, $bodyHTML);

163:  *

165:  *    be either WireMail() or a class descending from it. If a WireMail descending
166:  *    module is installed, it will be returned rather than WireMail():
167:  *
168:  *    $mail = wireMail();
169:  *    $mail->to('user@domain.com')->from('you@company.com');
170:  *    $mail->subject('Mail Subject')->body('Mail Body Text')->bodyHTML('Body HTML');
171:  *    $numSent = $mail->send();
172:  *
173:  *
174:  * @param string|array $to Email address TO. For multiple, specify CSV string or array.
175:  * @param string $from Email address FROM. This may be an email address, or a combined name and email address.
176:  *  Example of combined name and email: Karen Cramer <karen@processwire.com>
177:  * @param string $subject Email subject
178:  * @param string|array $body Email body or omit to move straight to $options
179:  * @param array|string $options Array of options OR the $bodyHTML string. Array $options are:
180:  *  body: string
181:  *  bodyHTML: string
182:  *  headers: associative array of header name => header value
183:  *  Any additional options will be sent along to the WireMail module or class, in tact.
184:  * @return int|WireMail Returns number of messages sent or WireMail object if no arguments specified.
185:  *
186:  */
