Class PagesType

ProcessWire PagesType

#pw-summary Provides an interface to the Pages class but specific to a given page class/type, with predefined parent and template. #pw-body = This class is primarily used by the core as an alternative to $pages, providing an API for other Page types like User, Role, Permission, and Language. The $users, $roles, $permissions and $languages API variables are all instances of PagesType. This class is typically not instantiated on its own and instead acts as a base class which is extended.
