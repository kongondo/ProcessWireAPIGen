 484:     /**
 485:      * Return an image (Pageimage) sized/cropped to the specified dimensions. 
 486:      * 
 487:      * `$thumb = $image->size($width, $height, $options);`
 488:      * 
 489:      * The default behavior of this method is to simply create and return a new resized version of the image,
 490:      * leaving the original in tact. Width and height of the target size are the the first two arguments. 
 491:      * The third argument called `$options` enables you to modify the default behavior of the size() method 
 492:      * in various ways. This method only creates the newly sized image once, and then it caches it. Future 
 493:      * calls simply refer back to the previously resized image. 
 494:      * 
 495:      * ~~~~~
 496:      * // Get an image to resize
 497:      * $image = $page->images->first();
 498:      * 
 499:      * // Create 400x300 thumbnail cropped to center
 500:      * $thumb = $image->size(400, 300); 
 501:      * 
 502:      * // Create thumbnail with cropping to top
 503:      * $thumb = $image->size(400, 300, 'north'); 
 504:      * 
 505:      * // Create thumbnail while specifying $options
 506:      * $thumb = $image->size(400, 300, [
 507:      *   'cropping' => 'north',
 508:      *   'quality' => 60, 
 509:      *   'upscaling' => false,
 510:      *   'sharpening' => 'medium'
 511:      * ]); 
 512:      * 
 513:      * // Output thumbnail
 514:      * echo "<img src='$thumb->url' />";
 515:      * ~~~~~
 516:      * 
 517:      * **About the $options argument**
 518:      * 
 519:      * - If given a *string*, it is assumed to be the value for the "cropping" option. 
 520:      * - If given an *integer*, it is assumed you are specifying a "quality" value (1-100). 
 521:      * - If given a *boolean*, it is assumed you are specifying whether or not you want to allow "upscaling".
 522:      * - If given an *array*, you may specify any of the options below together: 
 523:      * 
 524:      * **All available $options**
 525:      * 
 526:      *  - `quality` (int): Quality setting 1-100 (default=90, or as specified in /site/config.php).
 527:      *  - `upscaling` (bool): Allow image to be upscaled? (default=true).
 528:      *  - `cropping` (string|bool|array): Cropping mode, see possible values in "cropping" section below (default=center).
 529:      *  - `suffix` (string|array): Suffix word to identify the new image, or use array of words for multiple (default=none).
 530:      *  - `forceNew` (bool): Force re-creation of the image even if it already exists? (default=false).
 531:      *  - `sharpening` (string): Sharpening mode: "none", "soft", "medium", or "strong" (default=soft).
 532:      *  - `autoRotation` (bool): Automatically correct rotation of images that provide this info? (default=true)
 533:      *  - `rotate` (int): Rotate the image this many degrees, specify one of: 0, -270, -180, -90, 90, 180, or 270 (default=0).
 534:      *  - `flip` (string): To flip, specify either "vertical" or "horizontal" (default=none).
 535:      *  - `hidpi` (bool): Use HiDPI/retina pixel doubling? (default=false).
 536:      *  - `hidpiQuality` (bool): Quality setting for HiDPI (default=40, typically lower than regular quality setting). 
 537:      *  - `cleanFilename` (bool): Clean filename of historical resize information for shorter filenames? (default=false).
 538:      *  - `nameWidth` (int): Width to use for filename (default is to use specified $width argument).
 539:      *  - `nameHeight` (int): Height to use for filename (default is to use specified $height argument). 
 540:      *  - `focus` (bool): Should resizes that result in crop use focus area if available? (default=true). 
 541:      *     In order for focus to be applicable, resize must include both width and height. 
 542:      *  - `allowOriginal` (bool): Return original if already at width/height? May not be combined with other options. (default=false)
 543:      * 
 544:      * **Possible values for "cropping" option**  
 545:      * 
 546:      *  - `true` (bool): Auto detect and allow use of focus (default).
 547:      *  - `false` (bool): Disallow cropping. 
 548:      *  - `center` (string): to crop to center of image.
 549:      *  - `x111y222` (string): to crop by pixels, 111px from left and 222px from top (replacing 111 and 222 with your values).
 550:      *  - `north` (string): Crop North (top), may also be just "n".
 551:      *  - `northwest` (string): Crop from Northwest (top left), may also be just "nw".
 552:      *  - `northeast` (string): Crop from Northeast (top right), may also be just "ne".
 553:      *  - `south` (string): Crop South (bottom), may also be just "s".
 554:      *  - `southwest` (string): Crop Southwest (bottom left), may also be just "sw".
 555:      *  - `southeast` (string): Crop Southeast (bottom right), may also be just "se".
 556:      *  - `west` (string): Crop West (left), may also be just "w".
 557:      *  - `east` (string): Crop East (right), may alos be just "e".
 558:      *  - `blank` (string): Specify a blank string to disallow cropping during resize.
 559:      *  - `array(111,222)` (array): Array of integers index 0 is left pixels and index 1 is top pixels.
 560:      *  - `array('11%','22%')` (array): Array of '%' appended strings where index 0 is left percent and index 1 is top percent.
