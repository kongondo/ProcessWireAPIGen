ProcessWire ApiGen Documentation

This is personal project of Francis Otieno (kongondo) to automate the generation of PHP docs for the CMF/CMS ProcessWire API at regular cycles. The project files and more information can be found here. I am still working on this page as you can see :-)

Important

Please note that these are technical documentation not aimed at the average ProcessWire user but developers who are curious or need to find out in more detail about a specific ProcessWire class or function and/or use this as a reference. Most of the technical stuff here steps outside ProcessWire's public API into the inner workings of the CMF/CMS. If you would like to know more about such stuff, then this is for you. For all other use, please use the regular ProcessWire API docs and the ProcessWire Cheatsheet.
